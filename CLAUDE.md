# Handshake — Working Protocol (read this every session)

**Before doing anything in this project — including proposing a plan — read
[`docs/PROGRESS.md`](docs/PROGRESS.md).** It is the single source of truth for what is
already done, what is decided, and what is next. This prevents redoing completed work or
re-asking settled questions when the chat gets long or context resets.

Rules:
- Treat "Locked Decisions" in `docs/PROGRESS.md` as settled. Do not re-litigate them.
- Treat "Done"/checked milestones as complete. Do not rebuild or re-verify without reason.
- **After completing meaningful work, update `docs/PROGRESS.md`**: append a dated entry to
  the Session Log and refresh "Current Status" and "Next Up". Commit it with the work.
- Multi-machine setup (desktop + laptop) is documented in [`docs/SETUP.md`](docs/SETUP.md).
